"Do I need to know HTML/CSS to use this?"

You should be comfortable editing an HTML file to replace the placeholder content with your own name, projects, and links. If you know how to change text inside an <h1> tag and swap out a URL, you're good. The customization guide shows you exactly what to change with annotated screenshots. If you've never touched HTML before, you might need 30 minutes and a friend who can help — but the template is designed to be as simple as possible.

"Is this a React/Next.js template? Will it work with my framework?"

BentoFolio is pure HTML and CSS by design. That means no framework lock-in, no `node_modules`, and no build step. You can absolutely take the CSS and HTML structure and adapt it to React, Vue, Svelte, or any framework you work with. The design system (CSS variables, grid layout) ports cleanly into any component-based architecture.
