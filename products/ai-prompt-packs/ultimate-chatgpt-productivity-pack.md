## Category 1: 商业与创业 (25 Prompts)

### 1.1 商业计划生成器
```
你是一位资深商业顾问。帮我为以下商业想法写一份简洁的商业计划书：
- 产品/服务: [描述你的产品]
- 目标市场: [描述目标客户]
- 核心优势: [你的独特卖点]
- 当前阶段: [idea/prototype/已上线]

请包含：执行摘要、市场分析、收入模型、3个月里程碑、风险评估。用中文输出，简洁实用。
```

### 1.2 竞品分析矩阵
```
作为市场研究专家，分析以下行业的主要竞争对手。对每个竞品列出：
1. 核心功能
2. 定价策略
3. 目标用户
4. 优势(3个)
5. 劣势(3个)
6. 市场定位

行业: [输入行业]
竞品列表: [输入竞品名称，用逗号分隔]

输出一个对比表格 + 关键洞察和建议。
```

### 1.3 定价策略顾问
```
你是一位定价策略专家。帮我分析以下产品的定价方案：
产品描述: [描述]
成本结构: [固定成本 + 可变成本]
目标利润率: [%]
竞品价格范围: [从-到]

请给出3种定价方案(低价/中价/高价)，每种说明利弊，并给出你的推荐方案及理由。
```

### 1.4 用户画像创建

```
为目标市场创建3个详细的用户画像(Persona)。每个包含：
```

```
写一个5封邮件的销售序列，用于[产品/服务]的冷启动推广：

- 第2封: 提供价值(分享行业洞察)

每封邮件主题行 + 正文。语气: [正式/友好/专业]。
```

## 附赠: ChatGPT 使用技巧

### 高级技巧 #1: 链式提示
先让 AI 理解上下文 → 再提出具体要求 → 最后要求迭代优化

### 高级技巧 #2: 角色设定公式
"你是一位 [角色]，拥有 [经验年限] 的 [领域] 经验。你的任务是 [任务]。使用 [风格] 的风格，面向 [受众]。"

### 高级技巧 #3: 输出格式控制
始终指定输出格式: 表格/列表/代码块/JSON/Markdown
